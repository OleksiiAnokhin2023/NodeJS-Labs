# TaskMate - менеджер студентських завдань

## Опис ідеї

TaskMate - це веб-застосунок для управління навчальними завданнями студентів.
Мета проєкту - допомогти користувачам планувати свою навчальну діяльність, контролювати дедлайни та організовувати завдання за предметами.

## Основні можливості

- створення, редагування та видалення завдань
- встановлення дедлайнів
- позначення статусу виконання
- групування завдань за предметами
- перегляд найближчих дедлайнів
- фільтрація завдань (виконані/невиконані/прострочені)

## Цільова аудиторія

Студенти, які хочуть організувати свій навчальний процес, контролювати дедлайни та бачити прогрес у виконанні завдань.

## Формат виконання

Проєкт виконується індивідуально (без команди).
Усі подальші лабораторні роботи будуть реалізовані в межах цього застосунку.

## Технології (попередньо)

- Backend: Node.js / Express (або інший фреймворк на вибір)
- Frontend: React
- База даних: PostgreSQL
- Інструменти CI/CD: GitHub Actions
